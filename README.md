# Inventory Tracker
**Wails v2 + React + SQLite — Offline Windows Desktop App**

---

## Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| Go | 1.21+ | https://go.dev/dl/ |
| Node.js | 18+ | https://nodejs.org |
| GCC (for sqlite3) | any | [TDM-GCC](https://jmeubank.github.io/tdm-gcc/) or MSYS2 |
| Wails CLI | v2.9+ | `go install github.com/wailsapp/wails/v2/cmd/wails@latest` |

> **Windows note:** `go-sqlite3` requires CGO. Install TDM-GCC and ensure
> `gcc` is on your PATH (`gcc --version` should work in a terminal).

## Getting Started

### 1. Install Go dependencies
```bash
cd inventory-app
go mod tidy
```

### 2. Install frontend dependencies
```bash
cd frontend
npm install
cd ..
```

### 3. Run in development mode
```bash
wails dev
```
This starts a hot-reload dev server. The app opens as a native window.
The `wailsjs/` bindings are **auto-regenerated** here — delete the stub `App.js` first:
```bash
del frontend\wailsjs\go\main\App.js   # Windows
wails dev
```

### 4. Build for production (Windows `.exe`)
```bash
wails build
```
Output: `build/bin/inventory-app.exe`

The SQLite database (`inventory.db`) is created automatically next to the `.exe` on first launch.

## Features

| Feature | Details |
|---------|---------|
| **Add item** | Name, category, quantity, price, notes |
| **Edit item** | Click Edit on any row → pre-filled modal |
| **Delete item** | Click Delete → confirmation dialog |
| **Search** | Live search by name or category (debounced) |
| **Stats bar** | Real-time SKU count, total units, total inventory value |
| **Low stock highlight** | Qty = 0 → red, Qty < 5 → orange |
| **Offline** | 100% local SQLite, no network required |

---

## Database

SQLite file location: **same directory as the `.exe`** (or next to `main.go` during dev).

Schema:
```sql
CREATE TABLE items (
  id       INTEGER PRIMARY KEY AUTOINCREMENT,
  name     TEXT    NOT NULL,
  category TEXT    NOT NULL DEFAULT '',
  quantity INTEGER NOT NULL DEFAULT 0,
  price    REAL    NOT NULL DEFAULT 0.0,
  notes    TEXT    NOT NULL DEFAULT ''
);
```

---

## Extending

- **Add more fields**: extend the `Item` struct in `app.go`, update the SQL schema and the React form in `App.jsx`.
- **Export to CSV**: add a `ExportCSV() string` method on `App` and bind it.
- **Categories from DB**: add a `categories` table and a `GetCategories()` binding.
